# HandoffMed
Passagem de Plantão gerada por IA, simples, objetivo e de fácil compreensão. Ideal para evoluções muito grandes e pouco objetivas.


HandoffMed

Sistema de apoio à passagem de plantão médico utilizando IA.

Objetivos:

- Padronização da passagem de plantão.
- Resumo estruturado de evoluções médicas.
- Organização por leitos.
- Exportação TXT.
- Sem armazenamento permanente.
